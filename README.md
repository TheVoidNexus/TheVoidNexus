Hey, I'm the person behind [NullRift](https://github.com/nullrift), one-person dev organization working on various projects.

I mostly build things that matter to me: websites, desktop apps, plugins and more.

## My Preferences

- TypeScript pretty much everywhere. Runtime type errors have burned me enough already.
- Tailwind because writing CSS by hand in 2026 is a personal choice I've stopped making.
- Vite over anything else for frontend (I'm not waiting 1 minute for a dev server to start)
- Tauri > Electron. Chromium does not need to ship with my notes app.
