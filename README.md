## My philosophy
I’ve made some opinions over time:

### <img src="https://skillicons.dev/icons?i=ts" width="18" /> TypeScript - Why aren't you everywhere?
Runtime errors are the worst. TypeScript helps catch them before they happen, and I like that.
##

### <img src="https://skillicons.dev/icons?i=tailwind" width="18" /> Tailwind CSS - Thank You!
Writing native CSS in 2026 is completely outdated. Tailwind probably saved me hundreds of hours of searching through style.css files.

##

### <img src="https://skillicons.dev/icons?i=vite" width="18" /> Vite - How's it so fast?
If my dev server takes longer than a minute to start, I’m losing interest.

##

### <img src="https://skillicons.dev/icons?i=tauri" width="18" /> Tauri - Take that Electron!
Even though all big companies decided to run their "native apps" on Electron for convenience reasons; I'm out! Tauri is the future of desktop apps.
